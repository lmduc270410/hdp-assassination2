Hướng dẫn sử dụng

Dùng ngay trong trình duyệt
1) Dán link: chuyển sang tab Link, nhập URL rồi bấm Lấy nội dung từ link. Nếu website chặn truy cập trực tiếp, hãy dùng tab Dán đề hoặc File / Folder.
2) Dán đề: chuyển sang tab Dán đề, dán toàn bộ nội dung đề bài. Nếu một khối có nhiều bài, tool sẽ thử tách thành nhiều file.
3) Upload file hoặc folder: chuyển sang tab File / Folder, kéo thả hoặc chọn nhiều file. Nếu là folder chứa nhiều đề, tool sẽ sinh nhiều markdown tương ứng.
4) Xuất kết quả: bấm Tải xuống. Một bài sẽ tải xuống file .md; nhiều bài sẽ xuất .zip chứa các file .md.
5) AI lời giải: thêm một hoặc nhiều model. Model nào lỗi hoặc bị 429 thì tool sẽ tự chuyển sang model khác. Chỉ một request cho một bài, không gom nhiều bài vào một request.
Lưu ý: với bài khó, phần lời giải tự sinh vẫn nên được kiểm tra thủ công trước khi lưu vào kho đề.
